FAQ & GitHub Pages

Apakah harga sudah termasuk hosting? +
Tidak. Harga yang Anda bayarkan adalah untuk jasa pembuatan desain dan penulisan kode. Anda akan menerima file `.html`, `.css`, dan `.js`. Hosting (tempat menyimpan file agar online) adalah tanggung jawab Anda.
Bagaimana cara membuat website online gratis? +
Anda bisa menggunakan GitHub Pages. Ini layanan hosting statis gratis dari GitHub. Cukup upload file yang kami berikan ke repository GitHub Anda, dan dalam 5 menit website Anda sudah online dengan domain `username.github.io`.
Apa bedanya Hosting Gratis (GitHub) vs Berbayar? +
GitHub Pages gratis dan cocok untuk portfolio, landing page, atau blog statis. Namun, jika Anda butuh fitur kompleks (database, login backend, toko online dinamis), Anda butuh hosting berbayar (VPS/Shared Hosting).
Apakah kode bisa diedit nanti? +
Tentu bisa. Kami memberikan source code yang "Clean Code" (mudah dibaca dan dimengerti). Jika Anda belajar HTML/CSS sedikit, Anda bisa mengganti teks atau gambar sendiri dengan mudah.
